Algoritimo escolhido: FIFO
É um algoritmo de escalonamento para estruturas de dados do tipo fila. Apresenta o seguinte
critério: o primeiro elemento a ser retirado é o primeiro que tiver sido inserido,
é um algoritmo de escalonamento não preemptivo que entrega a CPU os processos pela ordem
de chegada.

Para compilação do programa em Linux, basta executar a seguinte linha de código no terminal:
$ gcc -Wall vmm.c -o vmm
$ ./vmm random 10 < anomaly.dat
$ ./vmm fifo 10 < anomaly.dat

O programa não possui nenhum problema quanto a execução.
